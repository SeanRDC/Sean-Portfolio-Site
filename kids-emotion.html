An interactive educational application designed to help children ages 4-8 recognize, understand, and express their emotions through engaging games, colorful characters, and age-appropriate activities.

Project Overview

This project is a front-end development showcase designed to demonstrate my ability to build engaging, responsive marketing websites. I created the "Kids Emotion Learning App" as a fictional product brand to serve as the subject for this design and coding exercise.

The landing page simulates a real-world product launch, featuring a vibrant, child-friendly user interface intended to appeal to parents and educators. The goal was to build a complete web presence from scratch including branding, layout, and interactivity to showcase how I can translate a product concept into a functional, high-converting website.

While the mobile application itself does not exist, the landing page is fully functional. It demonstrates my proficiency in structuring information, implementing responsive design patterns, and creating a cohesive visual identity for a digital product.
